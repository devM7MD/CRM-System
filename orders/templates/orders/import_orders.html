CSV Format Requirements

Your CSV file must have the following columns (in order):

1 Order Code (optional)
2 Customer Name (required)
3 Mobile Number (optional)
4 Shipping Address (optional)
5 Product ID/Code (optional)
6 Quantity (required)
7 Price Per Unit (AED) (required)
8 Product Variant (optional)
9 Notes (optional)
10 Order Date (YYYY-MM-DD) (optional)

Important Notes

  • If Order Code is empty, a new code will be auto-generated
  • If Product ID is not found, the order will be created without a product link
  • Quantity must be greater than 0
  • Price cannot be negative
  • Customer name is required for all orders
  • Orders with existing order codes will be skipped

Example CSV Format

Header row:
ORDER_CODE,CUSTOMER_NAME,MOBILE,ADDRESS,PRODUCT_ID,QUANTITY,PRICE,VARIANT,NOTES,ORDER_DATE
Data row:
ORD-001,John Doe,+971501234567,Dubai UAE,PROD-001,2,99.99,Red,Urgent delivery,2024-01-15
Make sure your CSV file follows the format above
